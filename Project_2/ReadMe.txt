Requirements to compile:
	gcc c compiler
	g++ compiler

Compiling:
	g++ P2.cpp
		There will be 2 warnings - these can be ignored
		You must move the images into the main file for the program to run
			These can be found in the S1 and S2 files

Running
	./a.out

	It will ask for inputs
	These can be acquired using either the S1.txt or S2.txt files

	It will print out the Transformation and the Transpose, and write a pgm named image.
